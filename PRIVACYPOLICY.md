# LowResoCamera プライバシーポリシー

このプライバシーポリシーは、aoumiusi（以下「当方」といいます）が提供するAndroid向けアプリケーション「LowResoCamera」（以下「本アプリ」といいます）における、ユーザー情報の取得、利用に関する方針を定めるものです。

## 1. 取得する情報および取得方法

本アプリは、ユーザーの個人を特定できる情報（氏名、住所、電話番号、メールアドレスなど）を**一切取得しません。**

本アプリがアクセスする情報は以下の通りです。

| 情報の種類 | 取得方法 | 取得の目的 |
| :--- | :--- | :--- |
| **カメラ（画像データ）** | 端末のカメラ機能および外部ストレージへのアクセス権限 | 撮影機能の提供および撮影した写真（ユーザーの作成した画像データ）の保存のため |
| **ストレージ** | 端末の外部ストレージへのアクセス権限 | ユーザーが撮影・加工した写真データを端末内に保存するため |

本アプリが取得したユーザーの画像データは、**すべてユーザーの端末内でのみ処理・保存され、当方を含む外部に送信されることはありません。**

## 2. 情報の利用目的

本アプリがアクセスする情報は、以下の目的で利用します。

1.  カメラ機能を動作させ、写真のリアルタイムプレビューおよび撮影を行うため。
2.  ユーザーが設定した「解像度調整」「色数削減」「モノクロ変換」などの画像処理を行うため。
3.  処理済みの画像データをユーザーの端末の所定のフォルダ（`Pictures/LowResCamera`）に保存するため。

## 3. 情報の第三者への提供

本アプリは、ユーザーから取得した情報を、法令に基づき開示が義務付けられる場合を除き、**ユーザーの同意なく第三者に提供・開示することはありません。**

## 4. 外部送信・連携について

本アプリは、いかなる外部サーバーにも情報を送信しません。広告配信モジュール、解析ツール（Google Analytics等）、トラッキング機能などは**一切組み込まれていません。**

## 5. プライバシーポリシーの変更

当方は、法令の変更や本アプリの機能追加などにより、本プライバシーポリシーを改定することがあります。改定した場合は、本アプリ内またはGoogle Playストアの掲載情報をもって通知いたします。
